Получение данных с MPU9250
==========================

Проверяем IMU ``MPU9250``:

- акселерометр;
- гироскоп;
- магнитометр.

Порядок проверки
----------------

1. Инициализируйте IMU.
2. Считайте сырые значения по осям XYZ.
3. Проверьте реакцию на движение и поворот платы.

Ожидаемый результат
-------------------

- Значения меняются при вращении/наклоне.
- Нет постоянных нулей по всем каналам.
