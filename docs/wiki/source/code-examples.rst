Код: объяснение и примеры
=========================

В этом разделе собраны примеры программ для проверки, настройки и запуска
учебного CubeSat.

Материал построен по принципу от простого к сложному: сначала проверяются
отдельные модули, затем подсистемы, а после этого запускается полная миссия.
