Возможные ошибки и диагностика
==============================

Типовые проблемы
----------------

- устройство не найдено по I2C;
- ошибка инициализации SD-карты;
- нет связи по радио;
- GPS не ловит спутники;
- некорректные значения датчиков.

Быстрые проверки
----------------

1. Питание и общая земля.
2. Соответствие пинов прошивке ``main_full``.
3. Проверка шины I2C сканером.
4. Проверка SPI и CS линий (SD на ``D4``, nRF CSN на ``D10``).
5. Для nRF — стабильное ``3.3V``.
6. При прошивке — временно отключить GPS от ``D0``.
