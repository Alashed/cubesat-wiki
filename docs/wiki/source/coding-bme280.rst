Получение данных с BME280
=========================

Проверяем базовые измерения модуля ``BME280``:

- температура;
- давление;
- влажность.

Порядок проверки
----------------

1. Инициализация датчика по I2C.
2. Чтение параметров в цикле.
3. Вывод значений в ``Serial Monitor``.

Необходимые библиотеки
----------------------

Перед загрузкой скетча установите через ``Library Manager``:

- ``Adafruit BME280 Library``
- ``Adafruit Unified Sensor`` (зависимость)

Что считать нормой
------------------

- Значения не ``nan``.
- Температура и влажность меняются плавно.
- Давление в реалистичном диапазоне.
