Телеметрия и радиосвязь (nRF24L01+)
===================================

Проверяем радиоканал передачи телеметрии.

Что проверить
-------------

- инициализация RF24;
- передача пакета;
- приём на базовой станции;
- стабильность канала.

Параметры из проекта
--------------------

- ``main_full``: канал ``76``, pipe ``TBOY1``;
- ``main_esp32_nrf_rx``: приём ``TBOY1`` на канале ``76``.

Важно по питанию
----------------

``nRF24L01+`` питается только от ``3.3V``.

Необходимые библиотеки
----------------------

Перед загрузкой скетчей установите через ``Library Manager``:

- ``RF24`` (TMRh20)
