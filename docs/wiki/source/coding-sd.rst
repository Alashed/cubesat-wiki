Работа с памятью (microSD)
==========================

Проверяем модуль хранения данных:

- инициализация SD-карты;
- создание файла;
- запись строки телеметрии;
- чтение записанных данных.

Подключение в main_full
-----------------------

- ``CS`` карты: ``D4``
- Общая SPI-шина: ``D11``/``D12``/``D13``

Замечание по SPI
----------------

При одновременной работе SD и nRF24L01+ рекомендуется резистор ``470 Ом``
в линии MISO со стороны SD-модуля.
