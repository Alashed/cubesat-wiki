Подготовка среды разработки
===========================

1. Установите Arduino IDE.
2. Подключите Arduino Nano по USB.
3. Выберите плату: ``Tools -> Board -> Arduino Nano``.
4. Выберите процессор, если требуется вашей версией платы.
5. Выберите COM-порт: ``Tools -> Port``.
6. Установите библиотеки проекта (RF24, Wire, SD, датчики).

Мини-проверка
-------------

- Плата определяется в IDE.
- Порт не занят другими программами.
- Скетч компилируется без ошибок.
