Получение данных с GPS (GY-NEO-6M)
==================================

Проверяем GPS-модуль:

- координаты;
- время;
- количество спутников.

Важно
-----

В ``main_full`` GPS использует ``D0 (RX0)``. Во время прошивки Arduino
рекомендуется отключать линию ``TX`` GPS от ``D0``.

Порядок проверки
----------------

1. Подключите GPS и запустите тестовый скетч чтения NMEA.
2. Дождитесь ``fix`` на открытом воздухе.
3. Проверьте обновление координат и времени.

Необходимые библиотеки
----------------------

Перед загрузкой скетча установите через ``Library Manager``:

- ``TinyGPSPlus`` (библиотека парсинга NMEA)
- ``SD`` (обычно встроена в Arduino IDE)
