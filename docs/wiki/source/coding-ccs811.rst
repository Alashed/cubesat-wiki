Получение данных с CCS811
=========================

Проверяем модуль ``CCS811``:

- ``TVOC``;
- ``eCO2``.

Порядок проверки
----------------

1. Инициализация датчика.
2. Ожидание готовности новых данных.
3. Чтение и вывод в ``Serial Monitor``.

Замечания
---------

- После старта датчику нужна стабилизация.
- Сравнивайте динамику значений, а не только одно чтение.
